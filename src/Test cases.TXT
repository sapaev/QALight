1. Авторизация пользователя.
Предусловие:
Пользователь запустил браузер для тестирования.
Шаги:
1) Запускаем ссылку https://ek.ua/ua/
2) Нажимаем на кнопку "Увiйти" в хэдере в правом верхнем углу.
3) В блоке авторизации выбираем блок "E-mail"
4) В поле "E-mail або Логiн" вводим действующий логин пользователя.
5) В поле "Пароль" вводим пароль от учетной записи пользователя, логин которого введен в пункте 4.
6) Нажимаем на кнопку "Увійти".
Ожидаемый результат: Пользователь успешно авторизирован на домашнюю страницу. Пользователю достуна кнопка
Логаут и пользователю доступна кнопка "Профиль".




2. Регитсрация пользователю с валидными данными.
Предусловие:
Пользователь запустил браузер для тестирования.
Шаги:
1) Запускаем ссылку https://ek.ua/ua/
2) Нажимаем на кнопку "Увiйти" в хэдере в правом верхнем углу.
3) В блоке авторизации выбираем блок "Зарегистрироваться"
4) В поле "Имя" вводим имя пользователя (Валидация отсутствует)
5) В поле "E-mail" вводим почту пользователя (Валидация login@domain)
6) В поле "Пароль" вводим пароль пользователя (Валидация отсутствует)
7) Нажимаем на кнопку "Зарегистрироваться".
8) В сплыывающем окне нажимаем на кнопку "Подтвердить".
Ожидаемый результат: Пользователь успешно зарегистрировался. После успешной авторизации пользователь автоматичекси авторизирован.
Пользователю достуна кнопка
Логаут и пользователю доступна кнопка "Профиль".




3. Проверка логаута пользователя.
Предусловие:
Пользователь авторизирован на сайте https://ek.ua/ua/
Шаги:
1) В хэдэре сайта нажимаем на кнопку "Выйти"
Ожидаемый результат:

4. Проверка работы смены имени пользователя.
Предусловие:
Пользователь авторизирован на сайте https://ek.ua/ua/
Шаги:
1) Нажимаем на кнопку "Профиль"
2) Нажимаем на кнопку "Редактировать профиль"
3) Удаляем имя пользователя в поле "Ваш нік"
4) Вводим новое значение в данное поле (Валидация отсутствует)
5) Нажимаем на кнопку "Сохранить".
Ожидаемый результат: Пользователь перещшел на страницу "Результат редактирования". Пользователю доступен блок, который содержит информация о профиле, которая была успешно сохранена.
Ник пользователя успешно обновлен.

6. Проверка работы поиска списка товаров с помощью строки поиска.

В резуьтате поиска, список товаров соответсвует запросу( Тип товара-"монитор", марка-"самсунг")

8. Проверка работы посика списка товаров с помощью фильтра выбора брендов товаров.
Предусловие:
Пользователь авторизирован на сайте https://ek.ua/ua/
Шаги:
1) Наводим курсором мыши на любой блок (Пример: "Комп'ютери").
2) Нажимаем на сабкатегорию "Ноутбуки".
3) В фильтре выбора бредов выбираем любое колитчество чек-боксов.
4) Нажимаем на кнопку "Подобрать"
Ожидаемый результат: Пользователь перешел на страницу результата поиска. 
В резуьтате поиска, список товаров соответсвует запросу( Тип товара, выбранные марки в фильтре).



9. Проверка работы открытя карточки найденного товара.
Предусловие:
Пользователь авторизирован на сайте https://ek.ua/ua/
Шаги:
1) В поле поиска вводим полное название товара, которое есть в базе данных (Пример: "Монітор Samsung C24F390F").
2) Нажимаем на кнопку "Знайти"
3) Нажимаем на найденную карточку.
Ожидаемый результат: пользователь перешел на страницу карточки. Подьзовалю доступен ллок описания товара и название карточки товара.
